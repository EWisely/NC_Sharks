# NC_Sharks
Code used in NC_Sharks project [^1] for shark diet metabarcoding from fecal swabs vs. stomach contents. The laboratory protocol for creating these data can be found on protocols.io at DOI: dx.doi.org/10.17504/protocols.io.ewov1qxokgr2/v1.[^2] This github repo covers the analysis steps including:
1) demultiplexing the two metabarcoding primers we used: MiFish[^3] and Berry's crustacean primer (BerryCrust in our code)[^4]
2) removing the adapters from the raw data from the MiSeq machine using Adapterama iNext primers [^5] and Illumina overhang adapters[^6].
