# NC_Sharks
Code used in NC_Sharks project [^1] for shark diet metabarcoding from fecal swabs vs. stomach contents. This github repo covers the analysis steps including:
1) demultiplexing the two metabarcoding primers we used: MiFish[^2] and Berry's crustacean primer (BerryCrust in our code)[^3]
2) removing the adapters from the raw data from the MiSeq machine using Adapterama iNext adapters and primers [^4]
